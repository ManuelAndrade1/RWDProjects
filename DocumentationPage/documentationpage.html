Introduction

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. Its high-level built in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development, as well as for use as a scripting or glue language to connect existing components together. Python's simple, easy to learn syntax emphasizes readability and therefore reduces the cost of program maintenance.

Python supports modules and packages, which encourages program modularity and code reuse. The Python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.

    Declaring Variables

    Declaring variables in Python is pretty easy, you just give your variable a name, and you assingn its value to it. No need to specify the data type or anything. Take a look at the code below.

    number = 10
    name = "John"
    floating = 1.3

    Here we are declaring three different variables: number which is a variable with a data type int, name which has a data type string, and floating, which is a float type variable. Python understands all this without us needing to declare which type each variable is.

    Data types

    So far we've seen 3 different data types: int, float and string, which are almost present in all programming languages.

    However, Python has a few more data types which are quite common and useful. Let's take a look at them.

    [Lists]

    Lists are what you would call arrays in other languages. The cool thing about this data type is that lists are mutable, meaning you can add and remove items from it. Furthermore, Python lists can contain different data types inside the same list. For example:

    my_list = [1, 2, 4, "Tom", 1.3, "Lisa"]

    Here we've created a list with different data types inside of it. Now let's add a new item to the end of the list with the .append() method and we will also use the .remove() method to remove the first item from our list.

    my_list.append("George")
    my_list.remove(1)

    Lists are just wonderful. You can do a lot of different things with them. But we'll leave it here, otherwise this section would become endless.

    (Tuples)

    Tuples are somewhat like lists. However, the difference between lists and tuples is that the latter is immutable and unchangable, you can neither change their size or values inside of them. Let's see an example:

    numbers = (1, 2, 3)

    Note that the way to tell python we want to create a tuple is by using parentheses "()".

    {Dictionaries}

    The last data type we are going to talk about are dictionaries. A dictionary is a way to store key-value-pairs in Python. You can store values by assigning a unique key to them. See the example below.

    phonebook = {
    "John": 11_3535_4545,
    "Amanda": 11_2323_6060,
    "Nichole": 11_1212_2244
    }

    Dictionaries can be a good way to save the phonenumbers of your friends, for instance. Notice that you put the key on the left between inverted commas "", and then the value you want that key to hold.(Notice the {} to tell Python you want to create a dictionary.)
